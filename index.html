    • Modbus

      Communication protocol developed by Modicon Systems in a simple format. It is a form of data transmission between electronic devices. The device that needs data is called the Modbus Master (Client). The device that provides the required data is called Modbus Slave (Server). In a standard Modbus Network, there is a single Master but up to 247 Slaves, each with an ID. Specify it like a house number from 1 to 247 and the Master can write information to the Slave.
